# Refactor mode

```bash
promptbridge "نضف الكود دا من غير ما تغير اللوجيك" --mode refactor
```

Expected style:

```text
Refactor and improve this code.

Requirements:
- Preserve the existing behavior.
- Do not change the existing business logic.
- Avoid unnecessary architectural changes.
- Keep public APIs stable unless a change is necessary.
- Explain the main refactoring decisions.
```
