# Explain mode

```bash
promptbridge "اشرحلي الكود دا ببساطة" --mode explain --bilingual
```

Expected style:

```text
## English prompt

Explain how this code works in simple language.

Requirements:
- Use simple language.
- Explain the main flow step by step.
- Identify the most important functions and files.
- Mention any confusing or risky parts.
- Provide a short summary at the end.

## Arabic summary

المطلوب شرح الكود بلغة بسيطة، خطوة بخطوة، مع توضيح أهم الملفات أو الدوال ونقاط الحذر.
```
