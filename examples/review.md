# Review mode

```bash
promptbridge "راجع الكود وشوف فيه مشاكل performance أو missing tests" --mode review
```

Expected style:

```text
Review this code and provide actionable feedback.

Focus on:
- Correctness.
- Edge cases.
- Maintainability.
- Performance.
- Security risks.
- Missing tests.

Constraints:
- Do not modify the code yet.
- First provide findings with severity and suggested fixes.
```
