# Launch Checklist

Use this checklist before sharing PromptBridge Arabic publicly.

## Repository

- [x] README explains the project clearly.
- [x] CLI examples are present.
- [x] MIT license is present.
- [x] Contributing guide is present.
- [x] Security policy is present.
- [x] Changelog is present.
- [x] GitHub issue templates are present.
- [x] GitHub Actions CI is present.

## Package

- [x] `npm pack --dry-run` succeeds.
- [x] Browser extension TypeScript is checked by CI.
- [x] Browser extension build is covered by CI.
- [x] Browser extension zip is generated for tagged releases.
- [x] Browser extension can be loaded from the repository without a local build.
- [x] VS Code-compatible extension packages as VSIX.
- [ ] Package is published to npm.
- [ ] npm package page is checked after publish.

## GitHub

- [x] Version tags are pushed through `v0.9.0`.
- [ ] GitHub Release is created from the latest tag.
- [ ] Chrome Web Store item is submitted.
- [ ] VS Code Marketplace item is submitted.
- [ ] Open VSX item is submitted.
- [ ] Repository topics are added.
- [ ] Good-first issues are opened.

## Outreach

- [ ] Short demo post is prepared.
- [ ] Arabic developer communities are selected.
- [ ] Feedback collection issue or discussion is opened.
