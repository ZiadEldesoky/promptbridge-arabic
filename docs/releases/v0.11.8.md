# PromptBridge Arabic v0.11.8

This release improves Arabic review-feedback prompts without using paid translation services.

## Highlights

- Added a local deterministic intent parser for phrases such as `إيه رأيك في الشغل اللي معمول دا`.
- Auto mode now generates a focused completed-work review prompt with concrete feedback requirements.
- General mode translates the same request naturally as `Review the completed work and provide feedback`.
- Handles selected text that includes minor RTL or editor artifacts, while keeping the main Arabic intent clean.

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.11.8.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.11.8.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.11.8-macos.zip`

## Local Commands

```bash
npm run release:browser
npm run release:vscode
npm run release:macos
```
