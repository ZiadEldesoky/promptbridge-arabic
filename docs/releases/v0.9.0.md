# PromptBridge Arabic v0.9.0

This release adds the first IDE extension path.

## Highlights

- Added a VS Code-compatible extension under `extensions/vscode`.
- Added editor commands to convert selected Arabic prompt text, copy converted input, or insert converted input.
- Added IDE settings for prompt mode, bilingual output, and secret redaction.
- Added `npm run build:vscode`, `npm run package:vscode`, and `npm run release:vscode`.
- Added VSIX packaging through `@vscode/vsce`.
- Added CI checks for VS Code extension typechecking, build, and packaging.
- Added publishing docs for Chrome Web Store, VS Code Marketplace, and Open VSX.

## Try the IDE extension locally

```bash
npm install
npm run release:vscode
code --install-extension artifacts/promptbridge-arabic-vscode-v0.9.0.vsix
```

## Notes

- The IDE extension is local and deterministic.
- It does not require an API key.
- It does not call external AI services.
- Cursor and other VS Code-compatible editors should be tested with the generated VSIX because compatibility varies by editor and version.
