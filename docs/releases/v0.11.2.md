# PromptBridge Arabic v0.11.2

This release fixes local macOS menu bar helper permission stability after rebuilds.

## Highlights

- The macOS helper build now prefers an available Apple Development signing identity.
- Local rebuilds no longer rely only on an ad-hoc cdhash when a valid signing identity exists.
- This makes macOS Accessibility permission more stable when updating the helper installed at `/Applications/PromptBridgeArabicMenuBar.app`.
- Explicit `general` mode now returns direct English translation when the Arabic text is understood, instead of a meta-template.

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.11.2.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.11.2.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.11.2-macos.zip`

## Local Commands

```bash
npm run release:browser
npm run release:vscode
npm run release:macos
```
