# PromptBridge Arabic v0.10.0

This release adds an experimental native macOS menu bar helper for lower-friction selected prompt replacement.

## Highlights

- Added a `PB` macOS menu bar helper under `extensions/macos`.
- Added **Auto Replace Selected Arabic** mode for converting only the currently selected Arabic text while the helper is enabled.
- Added Accessibility-based selected text reading with copy/paste fallback for custom prompt boxes.
- Added macOS app packaging through `npm run release:macos`, including a bundled deterministic converter built from the TypeScript core.

## Artifacts

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.10.0.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.10.0.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.10.0-macos.zip`

## Install the macOS helper

```bash
npm install
npm run build
npm run release:macos
open artifacts/PromptBridgeArabicMenuBar-v0.10.0.app
```

Enable **Auto Replace Selected Arabic** from the `PB` menu bar item, then grant Accessibility permission when macOS asks.
