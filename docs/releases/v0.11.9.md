# PromptBridge Arabic v0.11.9

This release adds explicit update checking so users can discover newer builds as PromptBridge evolves.

## Highlights

- Added `promptbridge check-updates` for CLI users.
- Added **Check for Updates** to the macOS menu bar helper.
- The update check contacts GitHub Releases only when the user triggers it.
- The macOS helper shows whether the installed build is current, whether an update is available, or whether GitHub could not be reached.
- If a GitHub Release is not available yet, the checker falls back to the repository's latest tag.

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.11.9.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.11.9.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.11.9-macos.zip`

## Local Commands

```bash
npm run release:browser
npm run release:vscode
npm run release:macos
promptbridge check-updates
```
