# PromptBridge Arabic v0.13.0

This release improves first-use speed, mode selection, and visible value for PromptBridge Arabic.

## Highlights

- Added `promptbridge stats` for deterministic prompt statistics.
- Stats include input size, output size, rough token estimates, selected mode, preserved technical tokens, and redaction count.
- Added a floating **Convert** button in the browser extension when Arabic text is selected inside supported web AI prompt boxes.
- Added **PromptBridge: Convert Arabic Selection with Mode...** in VS Code-compatible editors.
- The IDE mode picker supports `auto`, `general`, `fix`, `refactor`, `review`, `tests`, `explain`, and `security`.
- Updated the GitHub and launch-kit screenshots with clean PNG captures for browser, editor, CLI, and workflow overview usage.
- No API keys, external AI services, telemetry, or automatic prompt upload were added.
