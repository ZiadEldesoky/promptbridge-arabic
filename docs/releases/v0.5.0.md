# PromptBridge Arabic v0.5.0

## Highlights

- Added CLI agent wrappers through `promptbridge run`.
- Added shell examples that let users keep typing normal agent commands while Arabic prompt arguments are converted automatically.

## Usage

Direct:

```bash
promptbridge run codex "ظبطلي الكود دا وخليه responsive"
```

Shell wrapper setup:

```bash
source examples/shell/promptbridge-agents.zsh
codex "ظبطلي الكود دا وخليه responsive"
```

## Notes

This solves low-friction usage for CLI agents. GUI apps still need selected-text replacement, clipboard automation, extensions, or a future input method.
