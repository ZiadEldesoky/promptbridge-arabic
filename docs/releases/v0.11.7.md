# PromptBridge Arabic v0.11.7

This release improves longer Arabic code-quality prompts without using paid translation services.

## Highlights

- Added a local deterministic parser for code-quality attributes such as organized, maintainable, clean, and secure.
- Full prompts such as `خلي الكود منظم وقابل للصيانة واهتم بالأمان بتاعه` now translate as `Make the code organized, maintainable, and secure`.
- Auto mode keeps the structured coding-agent prompt while using the cleaner natural interpretation.
- Duplicate context lines are reduced when a quality parser already handled the selected prompt.

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.11.7.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.11.7.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.11.7-macos.zip`

## Local Commands

```bash
npm run release:browser
npm run release:vscode
npm run release:macos
```
