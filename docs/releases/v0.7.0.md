# PromptBridge Arabic v0.7.0

This release makes the browser extension more practical for daily GUI workflows.

## Highlights

- Browser extension settings are now saved locally in browser storage.
- Popup, context menu, and keyboard shortcut flows share the same saved defaults.
- Users can save default mode, bilingual output, redaction, and focused-field fallback behavior.
- The extension can convert the focused prompt box when no text is selected.
- Added site-aware prompt box selectors for ChatGPT, Claude, Gemini, Cursor web tools, and generic editable fields.
- Added tests for browser settings normalization and site adapter selector behavior.
- Added a browser-specific TypeScript check for extension source files.

## Usage

```bash
npm install
npm run build:browser
```

Then load `extensions/browser` from `chrome://extensions` using Developer mode.

Inside a web AI app:

1. Write Arabic in the prompt box.
2. Either select the text or just keep the prompt box focused.
3. Use the extension popup, context menu, or `Command+Shift+Y` on macOS.

## Notes

- No API key is required.
- No external AI service is called.
- Redaction remains opt-in unless the user saves it as a browser extension default.
