# PromptBridge Arabic v0.8.0

This release makes the browser extension usable without asking users to build it locally.

## Highlights

- The built browser extension bundle is committed under `extensions/browser/dist`.
- Users can clone the repository and load `extensions/browser` directly from `chrome://extensions`.
- Added `npm run package:browser` to create a browser extension zip.
- Added `npm run release:browser` to rebuild and package the browser extension.
- Added a tag-based GitHub Actions workflow that uploads the browser extension zip to the GitHub Release.
- CI now checks that the committed browser extension bundle matches the source.

## Use without building

### From GitHub Releases

1. Download `promptbridge-arabic-browser-extension-v0.8.0.zip` from the release assets.
2. Unzip it.
3. Open `chrome://extensions`.
4. Enable Developer mode.
5. Choose **Load unpacked** and select the unzipped folder.

### From the repository

1. Clone or download the repository.
2. Open `chrome://extensions`.
3. Enable Developer mode.
4. Choose **Load unpacked** and select `extensions/browser`.

No `npm install` or build step is needed for extension users.
