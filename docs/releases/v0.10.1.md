# PromptBridge Arabic v0.10.1

This patch release fixes the experimental macOS menu bar helper visibility.

## Fixes

- Replaced the text-only `PB` menu bar title with a template text-bubble status icon.
- Added a first-run notice so users can find the PromptBridge icon after launch.
- Switched the helper to an explicit AppKit launcher so the app delegate stays alive reliably.

## Artifacts

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.10.1.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.10.1.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.10.1-macos.zip`
