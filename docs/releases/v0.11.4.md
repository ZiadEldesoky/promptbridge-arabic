# PromptBridge Arabic v0.11.4

This release improves selected-text translation for organized-code phrases.

## Highlights

- Full prompts such as `خلي الكود منظم` now become practical refactor guidance for organized, maintainable code.
- Short selected fragments such as `منظم` now translate as `Organized and maintainable`.
- The deterministic glossary now covers common Egyptian Arabic variants such as `مرتب`, `مرتبة`, `منظم`, and `منظمة`.
- Translation output avoids leaving Arabic fragments in handled organized-code phrases.

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.11.4.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.11.4.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.11.4-macos.zip`

## Local Commands

```bash
npm run release:browser
npm run release:vscode
npm run release:macos
```
