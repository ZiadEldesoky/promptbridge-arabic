# PromptBridge Arabic v0.11.1

This release makes selected-text conversion more practical for real everyday prompts by improving deterministic intent detection and natural Arabic-to-English handling.

## Highlights

- Short security hardening prompts such as "خلي الكود آمن" now become actionable security improvement prompts instead of generic fix templates.
- Short performance prompts such as "خلي الكود أسرع" now become targeted performance refactor prompts.
- Friendly or non-coding selected Arabic text can stay as natural English translation instead of being forced into a coding-agent template.
- The Egyptian Arabic glossary now covers more security, performance, testing, review, UI, auth, business, and friendly phrases.
- Glossary matching now handles attached Arabic conjunctions such as "وخليه" and "والعميل".

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.11.1.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.11.1.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.11.1-macos.zip`
