# PromptBridge Arabic v0.9.4

This patch release improves selected Arabic text replacement in macOS IDE chat inputs.

## Highlights

- Uses the native macOS pasteboard through `pbpaste` and `pbcopy` for focused input replacement.
- Polls briefly after sending copy so custom IDE chat inputs have time to update the clipboard.
- Improves the warning when PromptBridge cannot read selected Arabic text from the active input.

## Artifacts

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.9.4.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.9.4.vsix`

## Install from VSIX

```bash
code --install-extension artifacts/promptbridge-arabic-vscode-v0.9.4.vsix
```

For Antigravity or Cursor chat inputs on macOS, select the Arabic text and press `Cmd+Shift+Y`. If the editor was already open during installation, run **Developer: Reload Window** once.
