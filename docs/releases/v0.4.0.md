# PromptBridge Arabic v0.4.0

## Highlights

- Added `promptbridge replace-selection` for selected-text replacement on macOS.
- This enables a practical global-shortcut workflow:
  1. Select Arabic text.
  2. Run the shortcut.
  3. PromptBridge copies, converts, and pastes the English prompt back.

## Usage

```bash
promptbridge replace-selection --redact
```

Optional clipboard restore:

```bash
promptbridge replace-selection --redact --restore-clipboard
```

## macOS permissions

The app that runs the command needs Accessibility permission, for example Terminal, iTerm, Raycast, or Automator.
