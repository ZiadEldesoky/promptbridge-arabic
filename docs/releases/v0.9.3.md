# PromptBridge Arabic v0.9.3

This patch release fixes focused IDE chat input replacement in VS Code-compatible editors.

## Highlights

- Fixed extension activation when running **PromptBridge: Replace Selected Text in Focused Input**.
- Removed the bundled Node clipboard package from the VS Code-compatible extension runtime.
- Kept macOS focused input replacement local and deterministic through VS Code clipboard APIs plus system copy/paste.

## Artifacts

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.9.3.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.9.3.vsix`

## Install from VSIX

```bash
code --install-extension artifacts/promptbridge-arabic-vscode-v0.9.3.vsix
```

For Antigravity or Cursor chat inputs on macOS, select the Arabic text and press `Cmd+Shift+Y`. If the editor was already open during installation, run **Developer: Reload Window** once.
