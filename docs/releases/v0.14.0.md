# PromptBridge Arabic v0.14.0

This release improves how PromptBridge understands Arabic and Egyptian Arabic coding prompts locally before generating the structured English prompt.

## Highlights

- Added local intent scoring for automatic mode selection instead of first-match keyword checks.
- Added slot extraction for likely targets such as files, pages, buttons, forms, API routes, and auth flows.
- Added detected constraints such as preserving visual design, preserving business logic, avoiding deletion, and making the smallest safe change.
- Improved real Arabic prompt coverage for auth tests, security reviews, build errors, save/reload bugs, and responsive checkout constraints.
- Updated the browser extension and VS Code-compatible extension builds to use the improved core engine.
- No API keys, external AI services, telemetry, or automatic prompt upload were added.
