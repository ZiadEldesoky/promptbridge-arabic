# PromptBridge Arabic v0.9.1

This release prepares PromptBridge Arabic for browser and IDE extension marketplace submission.

## Highlights

- Added PNG icons for the browser extension and VS Code-compatible extension.
- Changed the browser extension to inject its conversion script only after a user action.
- Added a public privacy policy for extension marketplace review.
- Expanded Chrome Web Store, VS Code Marketplace, and Open VSX listing copy and reviewer instructions.
- Updated packaging checks so the browser extension zip includes icon assets.

## Artifacts

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.9.1.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.9.1.vsix`

## Install from VSIX

```bash
code --install-extension artifacts/promptbridge-arabic-vscode-v0.9.1.vsix
```
