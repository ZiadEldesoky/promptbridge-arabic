# PromptBridge Arabic v0.9.2

This release makes the VS Code-compatible extension faster to use inside editor documents and macOS IDE prompt boxes.

## Highlights

- Added `Cmd+Shift+Y` on macOS and `Ctrl+Shift+Y` on Windows/Linux to convert selected Arabic text in an editor.
- Added an editor right-click menu item: **PromptBridge: Convert Arabic Selection**.
- Added **PromptBridge: Replace Selected Text in Focused Input** for macOS IDE chat inputs and other focused text boxes.
- Documented that some AI chat inputs inside IDEs are custom workbench UI and may require focused selection replacement or clipboard mode.

## Artifacts

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.9.2.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.9.2.vsix`

## Install from VSIX

```bash
code --install-extension artifacts/promptbridge-arabic-vscode-v0.9.2.vsix
```

For Antigravity or Cursor chat inputs on macOS, select the Arabic text and press `Cmd+Shift+Y`. The extension uses system copy/paste to replace the selected Arabic prompt with the generated English prompt in the same input.
