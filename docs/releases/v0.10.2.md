# PromptBridge Arabic v0.10.2

This release makes PromptBridge more practical for everyday prompt writing and stabilizes the macOS menu bar helper.

## Highlights

- Added a `general` prompt mode for friendly Arabic text and business/product requests.
- Expanded the deterministic glossary with friendly phrases and product workflow terms such as customers, orders, shipping, payment, invoices, inventory, and dashboards.
- Improved Egyptian Arabic detection for preserving the current design, including "بدون ما تغير في الديزاين".
- Fixed a macOS helper crash caused by Swift regular-expression initialization.
- Built the macOS helper as a stable `PromptBridgeArabicMenuBar.app` bundle so Accessibility permissions remain more reliable across releases.
- Made the macOS helper prefer copy/paste fallback before Accessibility for better compatibility with Notes and IDE prompt boxes.

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.10.2.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.10.2.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.10.2-macos.zip`

## Local Commands

```bash
npm run release:browser
npm run release:vscode
npm run release:macos
```
