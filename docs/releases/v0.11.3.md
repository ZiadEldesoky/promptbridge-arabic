# PromptBridge Arabic v0.11.3

This release improves selected-text translation for compound code-quality phrases.

## Highlights

- Full prompts such as `خلي الكود آمن ونظيف` now become practical coding-agent guidance for secure, clean, and maintainable code.
- Short selected fragments such as `آمن ونظيف` now translate as `Secure, clean, and maintainable` instead of expanding into a full agent prompt.
- The deterministic glossary now covers common Egyptian Arabic variants for clean code and maintainability.
- Translation output avoids leaving Arabic fragments in the handled secure-and-clean code phrases.

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.11.3.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.11.3.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.11.3-macos.zip`

## Local Commands

```bash
npm run release:browser
npm run release:vscode
npm run release:macos
```
