# PromptBridge Arabic v0.12.1

This release improves update handling in the macOS menu bar helper.

## Highlights

- Added direct macOS update downloads from the update dialog.
- The menu bar helper now looks for the latest GitHub Release asset named like `PromptBridgeArabicMenuBar-v0.12.1-macos.zip`.
- If the macOS zip is available, users can click **Download Update** instead of opening the release page first.
- The downloaded zip is saved to the user's Downloads folder.
- After download, PromptBridge can reveal the zip in Finder.
- If a release asset is unavailable, the app still falls back to **Open Release**.

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.12.1.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.12.1.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.12.1-macos.zip`
