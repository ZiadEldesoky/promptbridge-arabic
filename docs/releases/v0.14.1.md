# PromptBridge Arabic v0.14.1

This patch improves the macOS menu bar helper's Accessibility permission behavior.

## Highlights

- Normal menu bar actions such as **Auto Replace Selected Arabic** and **Convert Current Selection** now check Accessibility permission silently.
- The macOS Accessibility prompt is now only requested from **Request Accessibility Permission**.
- This avoids repeated macOS permission dialogs when the helper is already installed but macOS needs the user to refresh the trusted app entry.
- No API keys, external AI services, telemetry, or automatic prompt upload were added.

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.14.1.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.14.1.vsix`
