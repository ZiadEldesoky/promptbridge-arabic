# PromptBridge Arabic v0.3.0

## Highlights

- Added clipboard automation with `promptbridge watch`.
- Added one-shot clipboard conversion with `promptbridge watch --once`.
- Improved practical usage across arbitrary AI apps without typing `promptbridge` before every prompt.

## Usage

Watch clipboard continuously:

```bash
promptbridge watch --redact
```

Convert current clipboard once:

```bash
promptbridge watch --once --redact
```

## Notes

This remains app-agnostic. Fully automatic replacement while typing inside arbitrary GUI applications still requires a future OS-level input method or app-specific integration.
