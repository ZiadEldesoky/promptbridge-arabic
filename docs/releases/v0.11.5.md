# PromptBridge Arabic v0.11.5

This release improves selected-text translation for combined organized-and-secure code requests.

## Highlights

- Full prompts such as `خلي الكود منظم وآمن` now become practical security and maintainability guidance.
- Direct translation mode returns `Make the code organized, maintainable, and secure`.
- Arabic Unicode is normalized before glossary matching, so decomposed input such as `وآمن` is handled like `وآمن`.
- Translation output avoids leaving Arabic fragments in handled organized/security phrases.

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.11.5.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.11.5.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.11.5-macos.zip`

## Local Commands

```bash
npm run release:browser
npm run release:vscode
npm run release:macos
```
