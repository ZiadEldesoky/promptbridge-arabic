# PromptBridge Arabic v0.11.6

This release improves selected-text translation for maintainability phrases.

## Highlights

- Full prompts such as `خلي الكود منظم وآمن وقابل للصيانة` now become practical secure maintainability guidance.
- Direct translation mode returns `Make the code organized, secure, and maintainable`.
- Short fragments such as `قابل للصيانة` now translate as `Maintainable`.
- Translation output avoids leaving Arabic fragments in handled maintainability phrases.

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.11.6.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.11.6.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.11.6-macos.zip`

## Local Commands

```bash
npm run release:browser
npm run release:vscode
npm run release:macos
```
