# PromptBridge Arabic v0.2.0

## Highlights

- Config file support.
- Custom glossary loading.
- Improved OSS packaging and documentation.
- Local deterministic CLI remains the core workflow.

## CLI

```bash
promptbridge "راجع الصلاحيات في middleware.ts" --config ./promptbridge.config.json
```

## Config

Supported config locations:

- `.promptbridge.json`
- `promptbridge.config.json`
- `~/.promptbridge/config.json`

Example:

```json
{
  "defaultMode": "fix",
  "defaultOutput": "english",
  "redactSecrets": true,
  "glossaryPath": "./promptbridge.glossary.json"
}
```
