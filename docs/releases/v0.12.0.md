# PromptBridge Arabic v0.12.0

This release makes the VS Code-compatible extension faster to use in editors such as VS Code, Cursor, and Antigravity.

## Highlights

- Added `Alt+Y` as the quick IDE shortcut.
- On macOS, this is shown to users as `Option+Y`.
- The shortcut works for selected editor text and focused IDE chat inputs.
- Kept `Cmd+Shift+Y` on macOS and `Ctrl+Shift+Y` on Windows/Linux as fallback shortcuts.
- Users can remap the shortcut from the editor's Keyboard Shortcuts if `Alt+Y` conflicts with their setup.
- No API keys, external AI services, telemetry, or automatic prompt upload were added.

## Local Commands

```bash
npm run release:vscode
code --install-extension artifacts/promptbridge-arabic-vscode-v0.12.0.vsix
```
