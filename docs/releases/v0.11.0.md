# PromptBridge Arabic v0.11.0

This release expands selected-text replacement beyond macOS so PromptBridge is more useful across everyday developer environments.

## Highlights

- Added experimental Windows and Linux support for `promptbridge replace-selection`.
- Extended VS Code-compatible focused input replacement to Windows and Linux.
- Added Windows helper scripts for PowerShell and AutoHotkey global shortcut workflows.
- Added a Linux helper script for desktop keyboard shortcuts.
- Documented platform requirements for macOS, Windows, Linux, browser, CLI, and editor workflows.

## Platform Notes

- macOS: stable selected-text replacement through Accessibility and copy/paste automation.
- Windows: experimental selected-text replacement through PowerShell input automation.
- Linux: experimental selected-text replacement through `xdotool` on X11 or `wtype` on Wayland.
- Browser extension and normal VS Code editor selection remain cross-platform.

## Assets

- Browser extension zip: `promptbridge-arabic-browser-extension-v0.11.0.zip`
- VS Code-compatible VSIX: `promptbridge-arabic-vscode-v0.11.0.vsix`
- macOS menu bar helper zip: `PromptBridgeArabicMenuBar-v0.11.0-macos.zip`

## Local Commands

```bash
npm run release:browser
npm run release:vscode
npm run release:macos
```
