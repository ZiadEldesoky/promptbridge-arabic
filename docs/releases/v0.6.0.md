# PromptBridge Arabic v0.6.0

This release expands PromptBridge beyond CLI-only workflows while keeping the deterministic TypeScript core reusable.

## Highlights

- Browser extension prototype for converting selected Arabic prompt text inside editable web prompt boxes.
- Browser popup controls for mode selection, bilingual output, and optional redaction.
- Context menu actions for normal conversion and conversion with redaction.
- Raycast script command for macOS selected-text replacement in GUI apps.
- New `npm run build:browser` command.
- CI now builds both the Node CLI and the browser extension bundle.

## How to try the browser extension

```bash
npm install
npm run build:browser
```

Then load `extensions/browser` from `chrome://extensions` using Developer mode.

## How to try the Raycast helper

Install and link the CLI:

```bash
npm install
npm run build
npm link
```

Then add `extensions/raycast` as a Raycast Script Commands directory and grant Raycast Accessibility permission in macOS settings.

## Notes

- No API key is required.
- No external AI service is called.
- Redaction is opt-in for the browser extension and enabled in the Raycast helper command.
- True conversion while typing with zero selection or shortcut still needs a future input method or deeper app-specific integration.
