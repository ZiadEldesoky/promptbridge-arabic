# Code of Conduct

PromptBridge Arabic aims to be useful for Arabic-speaking developers and welcoming to contributors from different backgrounds and dialects.

Expected behavior:

- Be respectful and specific.
- Assume good intent while keeping technical discussions concrete.
- Prefer actionable feedback over vague criticism.
- Avoid personal attacks, harassment, or discriminatory language.

Unacceptable behavior may be removed from issues, discussions, or pull requests.
